Adds handles around images so you can resize them live in Chrome, Safari and other webkit browsers!

== Description ==

Ever wonder why you can't resize images in the wordpress editior like you can in Firefox and Internet Explorer? Well now you can.  Just upload and activate this plugin and you should be set.

== Installation ==

1. Upload `image-resize.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can now resize images in the wordpress tinyMCE editor.
